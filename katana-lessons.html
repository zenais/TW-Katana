Twice linear

Consider a sequence u where u is defined as follows:
  1. The number u(0) = 1 is the first one in u.
  2. For each x in u, then y = 2 * x + 1 and z = 3 * x + 1 must be in u too.
  3. There are no other numbers in u.
Ex: u = [1, 3, 4, 7, 9, 10, 13, 15, 19, 21, 22, 27, ...]
Task:
Given parameter n the function dbl_linear (or dblLinear...) returns the element u(n) of the ordered sequence u (so, there are no duplicates). 
        function dblLinear(n) {
            let list = {1: true};
            let i = 0;
            let nextIndex = 1;
        
            while ((i/2)-1 <= n ){
                while(list[nextIndex] === undefined){
                    nextIndex += 1;
                }
                let val1 = (nextIndex * 2) + 1;
                let val2 = (nextIndex * 3) + 1;
                list[val1] = true;
                list[val2] = true;
                i += 1;
                nextIndex += 1;
            }
            i = 0;
                for (const prop in list){
                    if (i === n) return Number(prop);
                    i += 1;
                }
            return;
        }

Pyramid Slide Down

Imagine that you have a pyramid built of numbers, like this one here: 
    /3/
    \7\ 4 
    2 \4\ 6 
    8 5 \9\ 3
Let's say that the 'slide down' is the maximum sum of consecutive numbers from the top to the bottom of the pyramid. As you can see, the longest 'slide down' is 3 + 7 + 4 + 9 = 23 
    * With the input `[[3], [7, 4], [2, 4, 6], [8, 5, 9, 3]]`       
    * Your function should return `23`.                                
                            
        function longestSlideDown (pyramid) {

            for (let i = pyramid.length - 2; i >= 0 ; i--){
                for (let j = 0; j < pyramid[i].length; j++){
                    let sum2= 0;
                    let sum1 = 0;
                    sum1 = pyramid[i][j] + pyramid[i+1][j]; 
                    sum2 = pyramid[i][j] + pyramid[i+1][j+1]; 
                    pyramid[i][j] = sum1 > sum2 ? sum1 : sum2;
                }
            }
            return pyramid[0][0];
        }

Create Phone Number

Write a function that accepts an array of 10 integers (between 0 and 9), that returns a string of those numbers in the form of a phone number.
Example

createPhoneNumber([1, 2, 3, 4, 5, 6, 7, 8, 9, 0]) // => returns "(123) 456-7890"

The returned format must be correct in order to complete this challenge.
Don't forget the space after the closing parentheses! 
        function createPhoneNumber(numbers){
            let phoneNumber = "(xxx) xxx-xxxx"
            
            for (let i = 0; i < numbers.length; i++) {
                phoneNumber = phoneNumber.replace("x", numbers[i]);
            }
            return phoneNumber
            }

Stop gninnipS My sdroW!

Write a function that takes in a string of one or more words, and returns the same string, but with all five or more letter words reversed (Just like the name of this Kata). Strings passed in will consist of only letters and spaces. Spaces will be included only when more than one word is present.

Examples:

spinWords( "Hey fellow warriors" ) => returns "Hey wollef sroirraw"
spinWords( "This is a test") => returns "This is a test"
spinWords( "This is another test" )=> returns "This is rehtona test" 
        function spinWords(string) {
            let newArr = string.split(' ');
            for (let i = 0; i < newArr.length; i++) {
                if (newArr[i].length >= 5)
                newArr[i] = newArr[i].split('').reverse().join('');
            }
            return newArr.join(' ');
            }

BMI Calculator

Write function bmi that calculates body mass index (bmi = weight / height2).

if bmi <= 18.5 return "Underweight"
if bmi <= 25.0 return Normal"
if bmi <= 30.0 return "Overweight"
if bmi > 30 return "Obese" 
        function bmi(weight, height) {
            let x = weight
            let y = height
            let result = x / (Math.pow(y,2))
            if( result <= 18.5){
                return "Underweight";
                }else if (result <=25.0){
                return "Normal"
                }else if (result <= 30.0){
                return "Overweight"
                }else if(result > 30){
                return "Obese"
                }
            }

Localize The Barycenter of a Triangle

The medians of a triangle are the segments that unit the vertices with the midpoint of their opposite sides. The three medians of a triangle intersect at the same point, called the barycenter or the centroid. Given a triangle, defined by the cartesian coordinates of its vertices we need to localize its barycenter or centroid.

Examples:

barTriang([4, 6], [12, 4], [10, 10]) ------> [8.6667, 6.6667]
barTriang([4, 2], [12, 2], [6, 10]) ------> [7.3333, 4.6667]
 
        function barTriang(p1, p2, p3){
            var x = p1[0] + p2[0] + p3[0]
            var y = p1[1] + p2[1] + p3[1]
            var y1 = y / 3
            var x1 = x / 3
            var roundX = x1.toFixed(4)
            var roundY = y1.toFixed(4)
            return [Number([roundX]), Number(roundY)];
            }
